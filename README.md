# LocalApp

# Steps:
1- Clone App
<br />
2- cd LocalApp
<br />
3- yarn (Or npm install
<br />
4- cd ios , pod install
<br />
5- running application
<br />
  to run in android: react-native run-android
<br />
  for ios (Mac only): react-native run-ios
